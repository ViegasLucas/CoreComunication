# Análise Rápida

Faça uma análise usando o template `analysis-template` (em `.agents/skills/`) sobre o
tema informado.

Entrada do usuário: o texto após o comando descreve o requisito/tema a analisar.

```
<requirements>
[Entrada do usuário]
</requirements>
```

Coloque o resultado em `docs/alalisys/*.md`.
